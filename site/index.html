Native • SwiftUI • RTL-SDR

One console for radio, aircraft, weather, satellites, and decode workflows.

NeuralSDR2 is a native macOS SDR application built around a universal operations map, a live radio console, and an integrated library for recordings and decoded products.

Core workspaces

Built as a mission console, not just a tuner window.

Radio

Live tuning, demodulation, spectrum, waterfall, AGC, squelch, and recording.

Aircraft

Shared universal map for ADS-B context and dump978 weather overlay workflows.

Sat Ops

Pass prediction, Doppler-aware tuning, queued recording, and post-pass automation.

3D Earth

Orbital context and space situational awareness inside the same app shell.

Library

Saved captures, decoded NOAA products, packet reports, replay, and decode-again actions.

Internal workflows

Capture a pass, then keep working inside the app.

NeuralSDR2 now pushes beyond capture-only SDR use. NOAA APT recordings can flow into internal image artifacts, packet passes can generate in-app analysis, and the library gives you a single place to review, replay, and re-run supported workflows.
